# 반병현 위키백과 스타일 페이지 구현 계획

## 목표 설명
"대취타" 한국어 위키백과 페이지의 디자인을 시각적으로 모방하되, 내용은 "반병현"에 대한 정보로 채운 웹페이지를 제작합니다. 인물 요약, 정보 상자(Infobox), 그리고 생애, 경력, 저서, 방송 출연 등의 섹션이 포함됩니다.

## 사용자 검토 필요 사항
> [!NOTE]
> 위키백과의 표준 CSS 스타일(글꼴, 색상, 레이아웃)을 사용하여 룩앤필(Look similar feel)을 구현할 것입니다. 프로필 사진은 실제 사진을 생성할 수 없으므로 자리 표시자(Placeholder) 이미지나 적절한 아이콘을 사용할 예정입니다.

## 변경 제안

### 웹페이지 구조

#### [신규] [index.html](file:///c%3A/Users/needl/Desktop/example2/index.html)
- 메인/타겟 위키백과 레이아웃
    - **헤더**: 위키백과 로고(텍스트/SVG), 검색바, 사용자 링크(모양만 구현).
    - **사이드바**: 탐색 링크 (대문, 랜덤 문서 등).
    - **콘텐츠 영역**:
        - **제목**: "반병현" 및 밑줄.
        - **정보 상자(InfoBox)**: 우측 정렬된 표, 프로필 상세 정보 (출생, 국적, 학력, 직업, 소속 등).
        - **도입부**: 인물 요약 (KAIST 출신, CTO, 작가, 유튜버 등).
        - **목차(TOC)**: 문서의 목차.
        - **섹션**:
            1. 생애
            2. 경력
            3. 저서
            4. 방송 출연
            5. 여담
            6. 각주

#### [신규] [styles.css](file:///c%3A/Users/needl/Desktop/example2/styles.css)
- 위키백과 전용 스타일링:
    - 글꼴: 산세리프(Sans-serif) 계열 (위키백과 및 OS 기본 폰트 사용).
    - 색상: 흰색 배경, 검은색 텍스트, 파란색 링크, 연한 회색 테두리 등.
    - 레이아웃: 사이드바와 메인 콘텐츠를 나누는 그리드/플렉스박스 구조.
    - `infobox` 클래스에 대한 위키백과 특유의 스타일 적용.
    - 참고: 최신 위키백과 스타일(Vector 2022) 또는 기존의 익숙한 스타일을 적절히 혼합하여 구현합니다.

## 검증 계획

### 수동 검증
- 브라우저에서 `index.html` 파일을 엽니다.
- 레이아웃이 한국어 위키백과 페이지와 유사한지 확인합니다.
- 내용이 조사된 "반병현" 님의 정보와 일치하는지 확인합니다.
- 기본적인 반응형 동작(브라우저 크기 조절 시 깨지지 않는지)을 확인합니다.
